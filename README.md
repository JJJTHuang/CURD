# 前端简单模拟数据的增删改查

<p>此项目简单包括了数据表的基本操作---(添加数据 ， 删除数据 ， 修改数据 ， 数据查询 ， 按指定列数据大小排序(点击对应列名(除操作列)即可) , 动态分页)</p>

<p style="color:red">(本地运行可能会出现chrome下直接打开由于跨域无法获取到数据,用火狐才能显示数据,若您需要,可以把json文件直接转成数组,即可避免跨域带来的问题。)</p>
